# Intentionally Blank Child Theme

This is a **child theme** for [Intentionally Blank](https://wordpress.org/themes/intentionally-blank/).

## Requirements

- You **must** have the [Intentionally Blank](https://wordpress.org/themes/intentionally-blank/) theme installed and activated as the parent theme for this child theme to work.

## Installation

1. Download or clone this repository into your `wp-content/themes/` directory.
2. Make sure the parent theme, [Intentionally Blank](https://wordpress.org/themes/intentionally-blank/), is installed.
3. Activate the **Intentionally Blank Child** theme from your WordPress dashboard.

## About

This child theme is intended for customizations and enhancements on top of the minimal [Intentionally Blank](https://wordpress.org/themes/intentionally-blank/) theme.

It is specifically designed to be used as a frontend for WordPress being used as a headless CMS.
